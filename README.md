🛍️ NovaCart – E-Commerce Website
📌 About
NovaCart is a responsive e-commerce frontend website inspired by platforms like Amazon. It is built using HTML, CSS, and JavaScript and includes product listings, navigation between sections, a simple cart interface, and basic form validation. The project focuses on modern UI/UX design with interactive elements and smooth user experience.
✨ Features
Product listing page
Navigation between sections (Home, Products, About, Contact, etc.)
Simple cart-style interface
Login & Register forms
Basic form validation
Responsive and modern UI design
🛠️ Tech Stack
HTML
CSS
JavaScript
🎯 Purpose
To practice frontend development, UI/UX design, and build a basic e-commerce website structure.
